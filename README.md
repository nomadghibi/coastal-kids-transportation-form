# Coastal Kids Transportation Form

Digital version of the Coastal Kids Academy Transportation Agreement / Field Trip Permission form.

**Privacy:** All data stays in your browser. Nothing is sent to a server.

## Features

- Paper-style letter-size form matching the original document
- Digital signature pad
- Download as PDF (client-side, no upload)
- Print directly from browser
- Draft saving to browser localStorage
- No database, no login, no backend

## Setup

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Import the repo at vercel.com/new
3. Vercel auto-detects Next.js — click Deploy
4. No environment variables needed

Or with Vercel CLI:

```bash
npm i -g vercel
vercel
```
